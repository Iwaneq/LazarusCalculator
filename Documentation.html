Procedury i Funkcje

Przyciski liczb

Obsługiwane przez kod zdarzenia "OnClick" - po ich wywołaniu, kod sprawdza czy nie ma żadnej liczby już wpisanej w okienko kalkulatora (Edit1). Jeśli jakaś liczba już tam jest, to liczba, którą reprezentuje wciśnięty przycisk zostaje dodana zaraz obok poprzedniej. Jeśli nie, w okno kalkulatora po prostu zostaje wstawiana liczba reprezentowana przez wciśnięty przycisk.

procedure TForm1.Button1Click(Sender: TObject);
begin     
			if Edit1.Text = '0' then     
					Edit1.Text := Button1.Caption     
			else         
					Edit1.Text := Edit1.Text + Button1.Caption
end;

Przycisk “dodaj”

Obsługiwany przez kod zdarzenia "OnClick" - po jego wywołaniu, kod zapisuje wartość okna kalkulatora (Edit1) do zmiennej, ustawia obecny operator matematyczny na “+” i czyści okienko kalkulatora, dając możliwość wpisana drugiej liczby.

procedure TForm1.AddButtonClick(Sender: TObject);
begin
     Operand1 := Edit1.Text;
     Operators := '+';
     Edit1.Text := '0';
end;

Przycisk “równa się”

Obsługiwany przez kod zdarzenia "OnClick" - po jego wywołaniu, kod zapisuje wartość okna kalkulatora (Edit1) do zmiennej, po czym sprawdza obecny operator matematyczny. Na podstawie wykonuje działanie, konwertując zmienne na odpowiednie typy i wynik zapisuje w okienku kalkulatora. 

procedure TForm1.EqualsButtonClick(Sender: TObject);
begin
     Operand2 := Edit1.Text;
     case(Operators) of
     '+':
       Answer := floattostr(StrToFloat(Operand1) + StrToFloat(Operand2));
     '-':
       Answer := floattostr(StrToFloat(Operand1) - StrToFloat(Operand2));
     '*':
       Answer := floattostr(StrToFloat(Operand1) * StrToFloat(Operand2));
     '/':
       Answer := floattostr(StrToFloat(Operand1) / StrToFloat(Operand2));
     end;
     Edit1.Text := Answer;
end;

Przycisk “przecinek”

Obsługiwany przez kod zdarzenia "OnClick" - po jego wywołaniu, kod sprawdza czy w okienku kalkulatora znajduje się przecinek. Jeśli nie, to go dopisuje do końca liczby w okienku kalkulatora.

procedure TForm1.DecimalButtonClick(Sender: TObject);
begin
     if (Pos(',', Edit1.Text)<>0) then
     exit
     else
         Edit1.Text := Edit1.Text + DecimalButton.Caption;
end;

procedure TForm1.SquareButtonClick(Sender: TObject);

     Operand1 := Edit1.Text;
     Answer := floattostr(StrToFloat(Operand1) * StrToFloat(Operand1));
     Edit1.Text := Answer;

Obsługa fizycznej klawiatury

Obsługiwany przez kod zdarzenia "FormKeyDown" - po jego wywołaniu, kod sprawdza czy wciśnięty klawisz należy do klawiszy reprezentujących liczby - jeśli należy, to dodaje tę liczbę do okienka kalkulatora.

procedure TForm1.FormKeyDown(Sender: TObject; var Key: Word; Shift: TShiftState);
begin
  case(Key) of
  VK_1:
       Key := 1;
  VK_2:
       Key := 2;
  VK_3:
       Key := 3;
  VK_4:
       Key := 4;
  VK_5:
       Key := 5;
  VK_6:
       Key := 6;
  VK_7:
       Key := 7;
  VK_8:
       Key := 8;
  VK_9:
       Key := 9;
  VK_0:
       Key := 0;
  else
       exit;
  end;

  if Edit1.Text = '0' then
     Edit1.Text := floattostr(Key)
  else
      Edit1.Text := Edit1.Text + floattostr(Key)
end;
